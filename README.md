# 🚀 CI/CD Pipeline Demo

A full DevOps project demonstrating automated build, test, containerization, and deployment using Jenkins and Docker.

## 🔧 Tech Stack
- Node.js (Express)
- Docker
- Jenkins
- GitHub
- Render (Cloud Deployment)

## ⚙️ Features
- Health check endpoint (`/health`)

- Automated CI/CD pipeline
- Dockerized deployment

## 🔄 CI/CD Workflow
1. Code pushed to GitHub
2. Jenkins triggers pipeline
3. App is tested
4. Docker image is built
5. Image pushed to Docker Hub
6. Container deployed automatically
7. Health endpoint verified
